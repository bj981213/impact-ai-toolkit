公開資訊追蹤 Agent

依固定週期整理指定議題的政策、新聞、統計與原始連結。

適用情境

誰會用到？

企業 CSR／ESG基金會NPO

依固定週期整理指定議題的政策、新聞、統計與原始連結。

準備資料

開始前請備妥

  • 議題、地區與對象
  • 核准來源清單
  • 追蹤週期
  • 人工查核與發布角色

不能直接上傳

個資與機密提醒

只追蹤公開來源，不建立個人檔案、不蒐集私人社群資訊，也不繞過付費或登入限制。

三步上手

怎麼使用這個流程

  1. 依週期搜尋核准公開來源
  2. 去除重複並保留發布日期與原始連結
  3. 人工查核後才發布摘要

AI Agent 專用

執行設定與治理規格

運作模式：固定週期搜尋公開來源，人工查核後發布。

觸發條件

  • 每週或每月指定時間
  • 追蹤條件已核准
  • 公開來源可存取

允許的資料來源

  • 政府公開資料
  • 研究機構公開資料
  • 核准媒體與組織網站

狀態流程

  1. 等待週期
  2. 搜尋與去重
  3. 來源檢查
  4. 等待人工查核
  5. 發布或退回

允許執行

  • 搜尋公開網頁
  • 摘要可開啟內容
  • 保留原始連結與日期

禁止自動執行

  • 編造來源或連結
  • 繞過登入或付費限制
  • 蒐集私人社群與個人資料

必須人工核准

  • 來源可信度
  • 摘要是否忠於原文
  • 正式發布

例外與失敗處理

  • 來源無法開啟時排除
  • 內容衝突時並列差異
  • 重大敏感事件時轉人工

最低紀錄要求

  • 查詢條件與時間
  • 來源與擷取日期
  • 人工排除與修改紀錄

成效指標

  • 有效原始連結比例
  • 重複來源比例
  • 摘要人工修正率
