工作請求整理 Agent

收到核准來源的 電子郵件 或表單後，整理請求、期限與建議負責角色。

適用情境

誰會用到？

企業 CSR／ESG基金會NPO

收到核准來源的 電子郵件 或表單後，整理請求、期限與建議負責角色。

準備資料

開始前請備妥

  • 核准的信箱或表單
  • 部門分工規則
  • 期限判讀規則
  • 人工指派角色

不能直接上傳

個資與機密提醒

只能讀取指定信箱或表單欄位；不得讀取私人信箱、附件個資或自行回信。

三步上手

怎麼使用這個流程

  1. 新請求進入後抽取事項與期限
  2. 依規則提出分類與建議負責角色
  3. 由人工確認指派與回覆

AI Agent 專用

執行設定與治理規格

運作模式：新請求觸發，只產生分類草稿，人工確認後才能指派。

觸發條件

  • 指定信箱收到新信
  • 表單新增回覆
  • 內容符合工作請求條件

允許的資料來源

  • 核准信箱欄位
  • 核准表單欄位
  • 部門分工規則

狀態流程

  1. 收到請求
  2. 抽取欄位
  3. 分類與期限檢查
  4. 等待人工指派
  5. 完成或退回

允許執行

  • 摘要請求
  • 抽取期限
  • 建議分類與角色

禁止自動執行

  • 自行回信
  • 自行承諾期限
  • 讀取私人信箱或未授權附件

必須人工核准

  • 請求分類
  • 負責人指派
  • 任何對外回覆

例外與失敗處理

  • 期限不明時標示待確認
  • 附件無權限時停止
  • 涉及個資或金錢時轉人工

最低紀錄要求

  • 來源與收到時間
  • 抽取結果
  • 人工修改與指派紀錄

成效指標

  • 請求漏列率
  • 期限判讀正確率
  • 分類人工修改率
