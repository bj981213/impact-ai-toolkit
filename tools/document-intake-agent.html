個資與機密提醒

Agent 不得開啟未授權文件或改動正式檔；只處理核准範圍並遵守保存規定。

三步上手

怎麼使用這個流程

  1. 偵測核准資料夾的新文件
  2. 依規則產生分類與命名建議
  3. 人工核准後才移動或改名

AI Agent 專用

執行設定與治理規格

運作模式：新文件觸發，只提出建議，人工核准後才能變更檔案。

觸發條件

  • 指定資料夾新增文件
  • 文件在核准範圍
  • 檔案掃描完成

允許的資料來源

  • 文件名稱與核准中繼資料
  • 分類規則
  • 檔名與版本規則

狀態流程

  1. 偵測新文件
  2. 規則檢查
  3. 產生建議
  4. 等待人工核准
  5. 執行或退回

允許執行

  • 讀取核准中繼資料
  • 建議分類與檔名
  • 標示重複或版本衝突

禁止自動執行

  • 未核准即改名或移動
  • 覆寫或刪除檔案
  • 開啟未授權內容

必須人工核准

  • 分類與保存位置
  • 檔名與版本
  • 任何檔案變更

例外與失敗處理

  • 規則衝突時停止
  • 疑似重複時並列比較
  • 敏感度不明時轉人工

最低紀錄要求

  • 原檔名與位置
  • 建議與核准結果
  • 變更前後紀錄

成效指標

  • 分類正確率
  • 命名人工修改率
  • 未授權變更件數為 0
